## The brand in one paragraph

Estancia Cafetera is a 10-room boutique hotel built around a working coffee estate. Volcanic highlands, dark earth palette, hand-thrown ceramics, slubbed linen, copper that's lived in. The brand IS the typography — no standalone logo. Premium through patience, not polish. We call it _volcanic restraint_.

---

## Token system

Dark coffee brown base (`#2E1F15`) with warm clay and ember as accent moments. Light is punctuation. Dark is base.

- **Wordmark:** Archivo Black + SVG ink-bleed filter
- **Editorial:** Migra Light / Light Italic _(currently Cormorant Garamond as free placeholder)_
- **Body:** Söhne / Untitled Sans _(currently Inter as free placeholder)_
- **Mono:** DM Mono

See `index.html` Section 03 (Color) and Section 04 (Typography) for the full system.

## Built with

Brand foundation built across multiple iterations with Claude. Image direction brief built in a single focused build. Deployed as static HTML on Vercel — no framework, no build step. The whole thing renders from one folder.
